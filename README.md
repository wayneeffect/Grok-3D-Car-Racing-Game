# Grok 3D Car Racing Game

A lightweight, fully playable **3D car racing game** that runs directly in your browser. Built with **Three.js** in a single HTML file.

## ✨ Features
- Realistic car physics (acceleration, braking, steering, friction)
- Smooth third-person chase camera
- Speedometer and lap counter
- Simple oval-style track with boundaries
- Keyboard controls (WASD + Arrow keys)
- Press **R** to reset position
- Built entirely with vanilla Three.js (no external dependencies except the CDN)

## 🎯 Controls
| Key              | Action                  |
|------------------|-------------------------|
| **W / ↑**        | Accelerate             |
| **S / ↓**        | Brake / Reverse        |
| **A / ←**        | Steer Left             |
| **D / →**        | Steer Right            |
| **R**            | Reset Car Position     |
